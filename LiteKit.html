Lite Kit

The goal of this project was to track and follow an object at a safe distance. This was accomplished by using the PixyCam in conjunction with the STM32 and the Parallax Propeller Board. The STM32 retrieves all relevant signature blocks detected by the PixyCam before sending a simple encoded movement command to the Parallax Propeller Board through UART. The Parallax Propeller Board will then decode and send the appropriate signals to the motor drivers to track and follow the object.

The full codes and a detailed explanation of this project can be found in *my Github repository*.
