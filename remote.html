Arduino Remote-Controlled Robot

The objective of this project was to enable users to directly control a robot and have the robot stop automatically when an obstacle is too close. To achieve this, an Arduino Mega 2560 is interfaced with a bluetooth module, ultrasonic sensors and H-bridge motor drivers. The Arduino will first receive a keystroke input from the user, verify if the path is clear, and then move accordingly. If the path is not clear, the Arduino will trigger an automatic stop to prevent collisions.

The full codes and a detailed explanation of this project can be found in *my Github repository*.
