TM5-700

The objective of this project was to ensure the Agilex Limo is capable of autonomous navigation between waypoints. To achieve this, our team set a pre-determined localized point on the global map to precisely determine its position and orientation within the environment. Users are then prompted to input their desired waypoints.

Using this localization information and the specified waypoints, the Agilex Limo employs a global path planner to generate a safe and efficient route from its current location to the designated destination. Throughout its journey, the robot continually updates the local path planner to dynamically navigate around obstacles, ensuring a smooth and obstacle-free trajectory.

For seamless execution, the entire navigation process relies on the Robot Operating System (ROS). The system sends the user-defined goal to the move_base node, a crucial component of the ROS navigation stack. The move_base node takes charge of coordinating the various navigation tasks, including localization, path planning, and obstacle avoidance.

The full codes and a detailed explanation of this project can be found in *my Github repository*.
