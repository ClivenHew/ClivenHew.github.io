Autonomous Waypoint Navigation

Situation

  • Learn how Robotic Operating System (ROS) can be used to develop robotic applications.
  • Learn how to use the navigation stack to navigate from waypoint A to B.
  • Challenge: Using the AgileX Limo, autonomously navigate around a prototype operational environment.

Task

  • Familiarise different concepts found in the navigation stack.
    • Navigation stack setup
    • GMapping
    • Global Planner
    • Local Planner
    • Recovery Behaviours

Action

  • Went through the wiki tutorial on ROS to setup and utilize navigation on TurtleBot3.
  • Used GMapping to create a map of the prototype environment.
  • Subscribed to essential topics to enable autonomous navigation on the AgileX Limo.
  • Used NavClient to send goals to the AgileX Limo.
  • Adjusted different parameters in the costmap yaml file to finetune obstacle avoidance.

Result

  • Demonstrated autonomous movement capability on the AgileX Limo.
  • Learnt how to setup ROS on robotic devices and utilize the navigation stack.
